Book

Last year I published Let’s Go, a book about how to build production-ready web applications with Go.

The book guides you through the start-to-finish build of a complete working application — covering topics like how to structure your code, manage dependencies, create dynamic database-driven pages, and how to authenticate and authorize users securely.

You can see the complete table of contents and buy the book at lets-go.alexedwards.net.
